# Approved Resume Points

No bullet should appear in a generated resume unless it is represented in this folder.

Each point record should contain:

- approved phrasings, including when each phrasing is preferred
- technologies and role boxes the point supports
- acceptable section headers or heading sets where the point can live

Preferred file naming convention:

- `point__<theme>.md`

When a new bullet is accepted, add or update the matching point file before treating the wording as part of the reusable resume system.
